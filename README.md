
# COVID Info in Kotlin

This project created for learning kotlin. It uses open [COVID api](https://documenter.getpostman.com/view/8854915/SzS7R6uu?version=latest) for retrieving timeline data about COVID cases, deaths and recovered people.
It generates text graphics for each country and saves result to disk space.
